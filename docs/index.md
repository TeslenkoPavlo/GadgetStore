# Розробка мобільного застосунку для інтернет-магазину цифрової техніки та електроніки

Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра ([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)).

Керівник роботи – **Бабич Олександр Вікторович**.

---

## Завдання до дипломної роботи

Розробити мобільний застосунок для продажу сучасної цифрової техніки та електроніки з використанням технології React Native. Пріоритетом проєктування визначити концепцію Mobile First, що має забезпечити інтуїтивно зрозумілий інтерфейс користувача. Архітектуру системи побудувати на базі хмарних сервісів платформи Firebase та реалізувати повну інтеграцію платіжної системи LiqPay. Ключовою особливістю проєкту передбачити впровадження інтелектуального ШІ-консультанта, який допомагатиме користувачам у виборі товарів через діалоговий інтерфейс. Готовий програмний продукт має бути підготовленим до публікації на платформах дистрибуції мобільних застосунків.
