### ЗАВДАННЯ ДО ДИПЛОМНОЇ РОБОТИ

Продукт має являти собою мобільний застосунок для продажу електроніки, розроблений з використанням технології React Native. Слід подбати про зручний, привабливий, інтуїтивно зрозумілий інтерфейс користувача. Використати платформу Firebase, подбати про інтеграцію платіжної системи (LiqPay), передбачити певні «родзинки», які виділять магазин серед інших (ШІ-консультант, оригінальні підходи до побудови інтерфейсу тощо). Готовий продукт має бути підготовленим до публікації в магазинах мобільних застосунків.
